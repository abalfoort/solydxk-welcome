PlayOnLinux

SolydXK comes with a wide variety of software. However, if you cannot find the right software in our repository and you need to use MS Windows software, you can install it using PlayOnLinux. Not all software can run this way, but a large number of software is already supported.

By using this software you may make your system vulnerable to MS Windows viruses and malware. The risk of infection is not higher than under native MS Windows.

Make sure you install software from trusted parties only.
