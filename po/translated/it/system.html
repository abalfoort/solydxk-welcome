Applicazioni di sistema

Cliccando sul bottone installa tutte le applicazioni selezionate verranno instalalte sul tuo sistema.
VirtualBox

VirtualBox è un software per la virtualizzazione di hardware x86, principalmente ad uso server, desktop e embedded. Il pacchetto VirtualBox si installa su un sistema operativo ospitante come un'applicazione, tale applicazione permette di installare ulteriori sistemi operativi, ognuno dei quali definiti come Guest OS può essere avviato ed eseguito nel proprio ambiente virtuale.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client è un client per connessioni FTP, FTPS e SFTP veloce e affidabile con molte funzionalità ed un'intuitiva interfaccia grafica.
Homepage: https://filezilla-project.org
