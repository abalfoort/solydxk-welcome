Applicazioni di sistema

ClamAV

ClamAV è un motore antivirus libero di rilevare trojan, virus, malware e altre minacce dannose.
VirtualBox

VirtualBox è un software per la virtualizzazione di hardware x86, principalmente ad uso server, desktop e embedded. Il pacchetto VirtualBox si installa su un sistema operativo ospitante come un'applicazione, tale applicazione permette di installare ulteriori sistemi operativi, ognuno dei quali definiti come Guest OS può essere avviato ed eseguito nel proprio ambiente virtuale.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client è un client per connessioni FTP, FTPS e SFTP veloce e affidabile con molte funzionalità ed un'intuitiva interfaccia grafica.
Homepage: https://filezilla-project.org
Homepage: https://www.clamav.net
Samba

Samba fornisce servizi di file e stampa per vari clienti Microsoft Windows e può integrarsi con un Microsoft Windows Server domain, sia come Domain Controller (DC) che come membro del domain. Supporta sia Active Directory che i Microsoft Windows NT domain.
Homepage: https://samba.org
CSS

La maggior parte dei DVD commerciali sono cifrati con il sistema di protezione CSS (Content Scramble System) che tenta di restringere il numero software in grado di riprodurre DVD.

SolydXK non fornisce in modo predefinito i pacchetti necessari per riprodurre tali DVD criptati.

Cliccando sul pulsante "installa", i seguenti pacchetti verranno installati:

  • libdvdcss2
  • w32codecs oppure w64codecs

Nota legale:

Controllare con la normativa locale ed assicurarsi che l'utilizzo di libdvdcss2 sia legale nell'area attuale.
