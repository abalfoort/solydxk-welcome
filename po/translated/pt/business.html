Aplicações para negócios

Esta é uma lista de aplicações recomendadas e adequadas para as empresas.
Gnucash

GnuCash é um “software” de contabilidade financeira pessoal e de pequenas empresas.

Projetado para ser fácil de usar, mas poderoso e flexível, GnuCash permite que acompanhe as contas bancárias, ações, renda e despesas. Tão rápido e intuitivo de usar como um registo de cheques, é baseado em princípios de contabilidade profissional para assegurar livros equilibrados e relatórios precisos.
Homepage: http://www.gnucash.org
PostBooks

ERP de código aberto para vendas, contabilidade, operações e CRM num único sistema (contabilidade, vendas, CRM, compras, etc.).

Use esta informação para iniciar o PostBooks:

  • Nome de utilizador: admin
  • Senha: solydxk
  • Servidor: localhost
  • Porta: 5432
  • Base de dados: demo, inicio rápido ou vazio

Pode gerir a base de dados PostgreSql com pgAdmin III (Login: postgres / solydxk)
Homepage: https://xtuple.com/products/postbooks
ProjectLibre

ProjectLibre é um sistema de “software” de gestão de projetos. Ele pretende ser um substituto completo em área de trabalho para Microsoft Project.
Homepage: http://www.projectlibre.org
