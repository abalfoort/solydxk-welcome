Aplicações empresariais

Esta é uma lista de aplicações recomendadas para as empresas.
Gnucash

O GnuCash é um ‘software’ de contabilidade financeira pessoal e para pequenas empresas.

Concebido para ser fácil de utilizar, mas poderoso e flexível, o GnuCash permite-lhe controlar contas bancárias, ações, rendimentos e despesas. Tão rápido e intuitivo de utilizar como um livro de cheques, baseia-se em princípios contabilísticos profissionais para garantir livros equilibrados e relatórios precisos.
Homepage: http://www.gnucash.org
PostBooks

ERP de fonte aberta para vendas, contabilidade, operações e CRM num único sistema (contabilidade, vendas, CRM, compras, etc.).

Utilize esta informação para iniciar o Post Books:

  • Nome de utilizador: admin
  • Senha: solydxk
  • Servidor: localhost
  • Porta: 5432
  • Base de dados: demo, quickstart ou vazia

É possível manter as bases de dados PostgreSql com o pgAdmin III (Login: postgres / solydxk)
Homepage: https://xtuple.com/products/postbooks
ProjectLibre

O 'ProjectLibre' é um sistema de ‘software’ de gestão de projetos. Pretende ser um substituto completo para o Microsoft Project.
Homepage: http://www.projectlibre.org
