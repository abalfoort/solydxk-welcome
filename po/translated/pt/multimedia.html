Aplicações multimédia

Esta é uma lista de aplicações recomendadas para utilizadores domésticos.
Gimp

GIMP é o acrónimo de 'GNU Image Manipulation Program'. É um programa distribuído gratuitamente para tarefas como retoque de fotos, composição de imagens e criação de imagens.

Tem muitas capacidades. Pode ser utilizado como um simples programa de pintura, um programa de retoque de fotografias de qualidade especializada, um sistema de processamento em lote ‘online’, um renderizador de imagens de produção em massa, um conversor de formatos de imagem, etc.
Homepage: http://www.gimp.org
Inkscape

O Inkscape é um editor de gráficos vetoriais de código aberto semelhante ao Adobe Illustrator, Corel Draw, Freehand ou Xara X. O que distingue o Inkscape é a utilização de SVG (Scalable Vetor Graphics), uma norma W3C aberta baseada em XML, como formato nativo.
Homepage: https://inkscape.org
Shotwell / Digikam

O Shotwell é um organizador de imagens concebido para permitir a gestão de fotografias pessoais.

O Shotwell pode importar fotografias e vídeos diretamente de uma câmara digital. O Shotwell agrupa automaticamente as fotografias e os vídeos por data e suporta a colocação de etiquetas. As suas funcionalidades de edição de imagem permitem aos utilizadores endireitar, cortar, eliminar olhos vermelhos e ajustar os níveis e o equilíbrio de cores.

Para o SolydK é instalado o Digikam. Tem a mesma funcionalidade que o Shotwell
Homepage Shotwell: https://wiki.gnome.org/Apps/Shotwell
Homepage Digikam: https://www.digikam.org
Clementine

O Clementine é um leitor de música moderno e um organizador de bibliotecas.

Clementine é um leitor de música multiplataforma. É inspirado pelo Amarok 1.4, com foco numa “interface” fácil de usar e rápido a procurar e tocar a sua música.
Homepage Clementine: https://www.clementine-player.org
HandBrake

O HandBrake é uma ferramenta para converter vídeo de praticamente qualquer formato para uma seleção de codecs modernos e amplamente suportados.
Homepage: https://handbrake.fr
SoundConverter

O SoundConverter é o principal conversor de ficheiros áudio. Lê tudo o que o GStreamer consegue ler (Ogg Vorbis, AAC, MP3, FLAC, WAV, AVI, MPEG, MOV, M4A, AC3, DTS, ALAC, MPC, Shorten, APE, SID, MOD, XM, S3M, etc...) e escreve em ficheiros Opus, Ogg Vorbis, FLAC, WAV, AAC e MP3.
Homepage: http://soundconverter.org
Pitivi / Kdenlive

O Pitivi é um editor de vídeo gratuito com uma ‘interface’ de utilizador bonita e intuitiva, uma base de código limpa e uma comunidade fantástica.

Para o SolydK é instalado o Kdenlive. Tem a mesma funcionalidade que o Pitivi
Homepage Pitivi: http://www.pitivi.org
Homepage Kdenlive: https://kdenlive.org
Transmission

O Transmission é um cliente BitTorrent que apresenta uma ‘interface’ simples em cima de um back-end multiplataforma.
Homepage Transmission: https://www.transmissionbt.com
