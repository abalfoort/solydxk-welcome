Aplicações multimédia

Esta é uma lista de aplicações recomendadas e adaptadas para utilizadores domésticos.
Gimp

GIMP é um acrónimo para GNU Image Manipulation Program. É um programa de distribuição livre para tarefas como o retoque de fotos, composição de imagem e autoria de imagens.

Tem muitas capacidades. Pode ser usado como um simples programa de pintura, um programa de retoque fotográfico com qualidade de especialista, um sistema de processamento em carga “on-line”, renderização de imagem, produção em massa, um conversor de formato de imagem, etc.
Homepage: http://www.gimp.org
Inkscape

Inkscape é um editor de código aberto de gráficos vetoriais, semelhante ao Adobe Illustrator, Corel Draw, Freehand, ou Xara X. O que define a diferença do Inkscape é a sua utilização de Gráficos Vetoriais Escaláveis (SVG), é com base em XML aberto e padrão W3C, como formato nativo.
Homepage: https://inkscape.org
Shotwell / Digikam

Shotwell é um organizador de imagens projetado para fornecer uma gestão de fotos pessoais.

Shotwell pode importar fotos e vídeos diretamente de uma câmara digital. Shotwell agrupa automaticamente as fotos e vídeos por data, e suporta etiquetas. Os seus recursos de edição de imagens permitem aos utilizadores endireitar, cortar, eliminar olhos vermelhos e ajustar os níveis e equilíbrio de cores.

Para SolydK, Digikam está instalado. Ele tem a mesma funcionalidade que Shotwell
Homepage Shotwell: https://wiki.gnome.org/Apps/Shotwell
Homepage Digikam: https://www.digikam.org
Clementine

Clementine é um leitor de música moderno e organizador de biblioteca.

Clementine é um leitor de música multiplataforma. É inspirado pelo Amarok 1.4, com foco numa “interface” fácil de usar e rápido a procurar e tocar a sua música.
Homepage Clementine: https://www.clementine-player.org
HandBrake

HandBrake é uma ferramenta para conversão de vídeo de quase qualquer formato para a seleção de codecs modernos, amplamente apoiadas.
Homepage: https://handbrake.fr
SoundConverter

SoundConverter é o líder conversor de ficheiros de áudio. Ele lê tudo o que GStreamer pode ler (Ogg Vorbis, AAC, MP3, FLAC, WAV, AVI, MPEG, MOV, M4A, AC3, DTS, ALAC, MPC, Shorten, APE, SID, MOD, XM, S3M, etc ... ), e escreve para o Opus, Ogg Vorbis, FLAC, WAV, AAC, e ficheiros MP3.
Homepage: http://soundconverter.org
Pitivi / Kdenlive

Pitivi é um editor de vídeo gratuito com uma “interface” bonita e intuitiva, uma base de código limpa e uma comunidade fantástica.

Para SolydK o Kdenlive está instalado. Ele tem a mesma funcionalidade que Pitivi
Homepage Pitivi: http://www.pitivi.org
Homepage Kdenlive: https://kdenlive.org
Transmission

Transmission é um cliente BitTorrent, que apresenta uma “interface” simples em cima de uma multiplataforma de back-end.
Homepage Transmission: https://www.transmissionbt.com
