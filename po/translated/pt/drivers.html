Controladores de “hardware”

SolydXK vem com uma grande variedade de controladores pré-instalados. Para a maioria dos propósitos esses controladores são suficientes, mas em alguns casos, o melhor desempenho é fornecido pelos controladores proprietários do fabricante.

O botão de instalação irá abrir o aplicativo Configurações do Sistema SolydXK. Esta é uma ferramenta para ajudá-lo a instalar os controladores de “hardware”. Ele verificará o seu “hardware” e listará todos os controladores proprietários disponíveis que pode instalar. Você só precisa selecionar o controlador recomendado para o seu “hardware” e a aplicação fará o resto.

Atualmente, os controladores de Nvidia , AMD/ATI e Broadcom podem ser instalados. Em Configurações de Sistema de SolydXK também permite instalar o Kernel PAE para sistemas multi-core de 32 bits.
