Controladores de ‘hardware’

O 'SolydXK' vem com uma grande variedade de controladores pré-instalados. Para a maioria das finalidades, estes controladores são suficientes, mas em alguns casos é possível obter um melhor desempenho com os controladores proprietários do fabricante.

O botão de instalação abrirá a aplicação 'SolydXK System Settings'. Esta é uma ferramenta para o ajudar a instalar estes controladores de ‘hardware’. Verificará o seu ‘hardware’ e listará quaisquer controladores proprietários disponíveis que possa instalar. Só precisa de selecionar o controlador recomendado para o seu ‘hardware’ e a aplicação fará o resto.

Atualmente, podem ser instalados controladoresNvidia, AMD/ATI e Broadcom drivers can be installed. O 'SolydXK System Settings' também permite instalar o kernel PAE kernel para sistemas multicore de 32 bits.
