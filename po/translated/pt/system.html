Aplicações de sistema

ClamAV

O ClamAV é um motor antivírus de código aberto para detetar trojans, vírus, malware e outras ameaças maliciosas.
Qemu

O QEMU (Quick Emulator) é um ‘software’ de virtualização de código aberto que permite a emulação e virtualização de vários sistemas operativos e aplicações em diferentes plataformas de ‘hardware’.
Homepage: https://www.qemu.org
VirtualBox

O VirtualBox é um virtualizador completo de uso geral para ‘hardware’ x86, direcionado para utilização em servidores, computadores de secretária e sistemas incorporados. O pacote VirtualBox instala-se num sistema operativo anfitrião existente como uma aplicação; esta aplicação anfitriã permite que sistemas operativos convidados adicionais, cada um conhecido como "SO convidado", sejam carregados e executados, cada um com o seu próprio ambiente virtual.
Homepage: https://www.virtualbox.org
FileZilla

O FileZilla Client é um cliente FTP, FTPS e SFTP multiplataforma rápido e fiável, com muitas funcionalidades úteis e uma ‘interface’ gráfica intuitiva para o utilizador.
Homepage: https://filezilla-project.org
Homepage: https://www.clamav.net
Samba

O Samba fornece serviços de ficheiros e de impressão para vários clientes Microsoft Windows e pode integrar-se num domínio Microsoft Windows Server, quer como Controlador de Domínio (DC), quer como membro do domínio. Suporta tanto o Active Diretory como os dominios Microsoft Windows NT.
Homepage: https://samba.org
CSS

A maioria dos DVD comerciais são encriptados com CSS (Content Scramble System), que tenta restringir o ‘software’ que pode reproduzir um DVD.

O SolydXK não vem com os pacotes necessários pré-instalados para reproduzir estes DVDs encriptados.

Ao clicar no botão de instalação abaixo, os seguintes pacotes serão instalados no seu sistema:

  • libdvdcss2
  • w32codecs ou w64codecs

Aviso legal:

Verifique as leis locais para se certificar de que a utilização da libdvdcss2 é legal na sua área.
