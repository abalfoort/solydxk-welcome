Aplicações de sistema

Quando clicar no botão instalar, todas as aplicações seleccionadas serão instaladas no seu sistema.
VirtualBox

VirtualBox é um virtualizador completo de proposta geral para hardware x86 e x64, voltado para servidor, ambiente de trabalho e uso embutido. O pacote VirtualBox instala-se num sistema operativo anfitrião existente como um aplicativo; este aplicativo anfitrião permite que os adicionais sistemas operativos convidados, cada um conhecido como um sistema operativo cliente, para carregar e executar, cada um com seu próprio ambiente virtual.
Página inicial: https://www.virtualbox.org
FileZilla

FileZilla é um cliente FTP, FTPS e SFTP, multi-plataforma rápido e confiável com bastantes características úteis e uma interface de utilizador gráfica intuitiva.
Página inicial: https://filezilla-project.org
ClamAV

ClamAV is an open source antivirus engine for detecting trojans, viruses, malware & other malicious threats.
Homepage: https://www.clamav.net
Samba

Samba provides file and print services for various Microsoft Windows clients and can integrate with a Microsoft Windows Server domain, either as a Domain Controller (DC) or as a domain member. It supports both Active Directory and Microsoft Windows NT domains.
Homepage: https://samba.org
CSS

A maioria dos DVDs comerciais estão encriptados com CSS (Content Scramble System), que tenta restringir o software que pode ler um DVD.

SolydXK não vêm com os pacotes necessários pré-instalados para reproduzir esses DVDs encriptados.

Ao clicar no botão abaixo instalar os seguintes pacotes serão instalados no seu sistema:

  • libdvdcss2
  • w32codecs or w64codecs

Aviso legal:

Verifique com as leis locais para certificar-se do uso de libdvdcss2 se são legais na sua área.
