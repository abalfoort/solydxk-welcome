Aplicações de sistema

Quando clicar no botão instalar, todas as aplicações seleccionadas serão instaladas no seu sistema.
VirtualBox

VirtualBox é um virtualizador completo de proposta geral para hardware x86 e x64, voltado para servidor, ambiente de trabalho e uso embutido. O pacote VirtualBox instala-se num sistema operativo anfitrião existente como um aplicativo; este aplicativo anfitrião permite que os adicionais sistemas operativos convidados, cada um conhecido como um sistema operativo cliente, para carregar e executar, cada um com seu próprio ambiente virtual.
Página inicial: https://www.virtualbox.org
FileZilla

FileZilla é um cliente FTP, FTPS e SFTP, multi-plataforma rápido e confiável com bastantes características úteis e uma interface de utilizador gráfica intuitiva.
Página inicial: https://filezilla-project.org
