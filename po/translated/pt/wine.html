PlayOnLinux

SolydXK vem com uma ampla variedade de software. No entanto, se você não consegue encontrar o software certo no nosso repositório e você precisa usar o software MS Windows, você pode instalá-lo usando PlayOnLinux. Nem todos os softwares podem ser executados desta forma, mas um grande número de software já é suportada.

Ao utilizar este software você pode tornar o seu sistema vulnerável a vírus e malware MS Windows. O risco de infecção não é maior do que sob nativo MS Windows.

Certifique-se de instalar o software a partir de apenas partes confiáveis.
