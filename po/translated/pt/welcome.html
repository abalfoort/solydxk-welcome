Bem-vindo ao SolydXK

Obrigado por usar SolydXK

SolydXK é um sistema operativo de código aberto de ambiente de trabalho para empresas, organizações e utilizadores domésticos. Ambos SolydX e SolydK são baseados em Debian estável o que garante uma grande estabilidade e segurança.

Nós criamos uma lista de software recomendado para empresas e utilizadores domésticos que você pode instalar nas páginas seguintes.

Clique no botão Seguinte para começar.

Software

Todo o software é construído, mantido e controlado pela comunidade. Base de dados de software chamados de "repositórios" são usados para oferecer e manter mais de 60.000 pacotes de software. Assim, não há necessidade de descarregar software a partir de sitios duvidosos. Você simplesmente procura e instala o software que precisa com o Gestor de Software.
Suporte

Se você é novo no SolydXK ou sempre que você ficar bloqueado, há a comunidade SolydXK para ajudá-lo a resolver o seu problema. Você pode procurar o fórum para descobrir se o problema já foi resolvido ou iniciar um novo tópico para obter ajuda.
Fórum: https://forums.solydxk.com
