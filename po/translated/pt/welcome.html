Bem-vindo ao SolydXK

Obrigado por utilizar o SolydXK

O SolydXK é um sistema operativo de ambiente de trabalho, de código aberto para empresas, organizações e utilizadores domésticos. Tanto o SolydX como o SolydK são baseados no Debian stable para garantir a maior estabilidade e segurança.

Criámos uma lista de ‘software’ recomendado para empresas e utilizadores domésticos que pode instalar nas próximas páginas.

Clique no botão Seguinte para começar.

Software

Todo o ‘software’ é criado, mantido e verificado pela comunidade. As bases de dados de ‘software’ chamadas “repositórios” são utilizadas para oferecer e manter mais de 60.000 pacotes de ‘software’. Assim, não há necessidade de descarregar ‘software’ de sítios ‘Web’ duvidosos. Basta procurar e instalar o ‘software’ de que necessita com o Gestor de ‘software’.

Suporte

Se é novo no SolydXK ou se tiver dificuldades, existe a comunidade SolydXK para o ajudar a resolver o seu problema. Pode pesquisar no fórum para ver se o seu problema já foi resolvido ou iniciar um novo tópico para obter ajuda.
Forum: https://forums.solydxk.com
