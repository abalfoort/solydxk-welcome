Aplicaciones de negocios

Esta es una lista de aplicaciones recomendadas y pensadas para empresas.

Gnucash

GnuCash es un software diseñado, para la contabilidad personal o para la contabilidad de pequeñas empresas.

Diseñado para ser fácil de usar, pero potente y flexible, GnuCash le permite el manejo de cuentas bancarias, acciones, ingresos y gastos. Tan rápida e intuitiva como un registro de chequera, se basa en principios de contabilidad profesional para garantizar libros equilibrados y reportes precisos.
Homepage: http://www.gnucash.org
PostBooks

ERP de código abierto para ventas, contabilidad, operaciones y CRM en un único sistema (contabilidad, ventas, CRM, compras, etc.).

Utilice esta información para iniciar PostBooks:

  • Usuario: admin
  • Clave: solydxk
  • Servidor: localhost
  • Puerto: 5432
  • Base de datos: demo, quickstart o vacía

Puedes mantener las bases de datos de PostgreSql con pgAdmin III (Login: postgres / solydxk)
Homepage: https://xtuple.com/products/postbooks
ProjectLibre

ProjectLibre es un sistema completo de software de gestión de proyectos. Tiene la intención de ser una opción libre de escritorio, reemplazo del programa Microsoft Project.
Homepage: http://www.projectlibre.org
