Aplicaciones de sistema

ClamAV

ClamAV es un motor antivirus de código abierto para detectar troyanos, virus, malware y otras amenazas maliciosas.
VirtualBox

VirtualBox es un virtualizador completo de uso general para hardware x86, destinado a servidores, escritorios y aplicaciones integradas. El paquete de VirtualBox se instala en un sistema operativo anfitrión existente como una aplicación; esta aplicación anfitriona permite cargar y ejecutar sistemas operativos invitados adicionales, cada uno conocido como SO invitado, con su propio entorno virtual.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client es un cliente multiplataforma FTP, FTPS y SFTP rápido y fiable con muchas funciones útiles y una interfaz gráfica de usuario intuitiva.
Homepage: https://filezilla-project.org
Homepage: https://www.clamav.net
Samba

Samba proporciona servicios de archivo e impresión para varios clientes de Microsoft Windows y puede integrarse en un dominio de Microsoft Windows Server, ya sea como controlador de dominio (DC) o como miembro del dominio. Es compatible con los dominios de Active Directory y Microsoft Windows NT.
Homepage: https://samba.org
CSS

La mayoría de los DVD comerciales están cifrados con CSS (Content Scramble System), que intenta restringir el software que puede reproducir un DVD.

SolydXK no vienen con los paquetes necesarios pre-instalado para jugar estos DVDs encriptados.

Haciendo clic en el botón Instalar los siguientes paquetes serán instalados en su sistema:

  • libdvdcss2
  • w32codecs o w64codecs

Aviso legal:

Compruebe con sus leyes locales para asegurarse de que el uso de libdvdcss2 sería legal en su área.
