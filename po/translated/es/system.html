Aplicaciones de sistema

Cuando haga clic en el botón Instalar todas las aplicaciones seleccionadas se instalarán en su sistema.
VirtualBox

VirtualBox is a general-purpose full virtualizer for x86 hardware, targeted at server, desktop and embedded use. The VirtualBox package installs on an existing host operating system as an application; this host application allows additional guest operating systems, each known as a Guest OS, to load and run, each with its own virtual environment.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client es un cliente multiplataforma FTP, FTPS y SFTP rápido y fiable con muchas funciones útiles y una interfaz gráfica de usuario intuitiva.
Homepage: https://filezilla-project.org
ClamAV

ClamAV is an open source antivirus engine for detecting trojans, viruses, malware & other malicious threats.
Homepage: https://www.clamav.net
Samba

Samba provides file and print services for various Microsoft Windows clients and can integrate with a Microsoft Windows Server domain, either as a Domain Controller (DC) or as a domain member. It supports both Active Directory and Microsoft Windows NT domains.
Homepage: https://samba.org
CSS

La mayoría de los DVD comerciales están cifrados con CSS (Content Scramble System), que intenta restringir el software que puede reproducir un DVD.

SolydXK no vienen con los paquetes necesarios pre-instalado para jugar estos DVDs encriptados.

Haciendo clic en el botón Instalar los siguientes paquetes serán instalados en su sistema:

  • libdvdcss2
  • w32codecs o w64codecs

Aviso legal:

Compruebe con sus leyes locales para asegurarse de que el uso de libdvdcss2 sería legal en su área.
