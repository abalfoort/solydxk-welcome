Controladores de hardware

SolydXK viene con una amplia variedad de controladores preinstalados. Para la mayoría estos controladores son suficientes, pero en algunos casos el mejor rendimiento es proporcionado por los controladores propietarios del fabricante.

El botón de instalación abrirá la aplicación SolydXK System Settings. Esta es una herramienta que le ayudará a instalar estos controladores de hardware. Comprobará su hardware y mostrará los controladores privativos disponibles que puede instalar. Solo necesita seleccionar el controlador recomendado para su hardware y la aplicación hará el resto.

Actualmente Nvidia, AMD/ATI y Broadcom controladores pueden ser instalado. SolydXK System Settings también permite instalar la PAE kernel para multi-core de los sistemas de 32 bits.
