Bienvenido a SolydXK

Gracias por utilizar SolydXK

SolydXK es un sistema operativo de código abierto para empresas, organizaciones y usuarios domésticos. Ambos SolydK y SolydX están basadas en Debian estable para garantizar la mayor estabilidad y seguridad.

Hemos creado una lista de software recomendado para empresas y usuarios domésticos que se pueden instalar en el próximo par de páginas.

Haga clic en el botón Siguiente para empezar.

Software

Todo el software está construido, mantenido y controlado por la comunidad. Las bases de datos de software denominada "Repositorios" se utilizan para ofrecer y mantener más de 60.000 paquetes de software. Por lo tanto, no hay necesidad de descargar software desde sitios web dudosos. Usted sólo tiene que buscar e instalar el software que necesita con el Software Manager.
Ayuda

Si eres nuevo a SolydXK o cuando te quedas atascado, existe la comunidad SolydXK para ayudarle a solucionar su problema. Puede buscar en el foro para encontrar si su problema ya ha sido resuelto o iniciar un nuevo tema para obtener ayuda.
Foro: https://forums.solydxk.com
