PlayOnLinux

SolydXK viene con una amplia variedad de software. Si usted no puede encontrar el software adecuado en nuestro repositorio y necesita usar software de MS Windows, puede instalarlo usando PlayOnLinux. No todo el software puede funcionar de esta manera, pero un gran número de software ya es compatible.

Usando este software, usted puede hacer que su sistema sea vulnerable a virus y malware de MS Windows. El riesgo de infección no es mayor que bajo MS Windows nativo.

Asegúrese de instalar el software solamente de las partes de confianza.
