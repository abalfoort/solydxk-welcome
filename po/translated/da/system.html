Systemprogrammer

Når du trykker på 'Installér'-knappen vil alle de valgte programmer blive installeret på dit system.
VirtualBox

VirtualBox er en generel virtuel-maskine applikation til x86-pc'er. Den installeres på et eksisterende system og afvikles som et program. Programmet giver så mulighed for at køre andre operativsystemer på samme tid, i hvert deres vindue, som om de var installeret på en fysisk maskine.
Hjemmeside: https://www.virtualbox.org
FileZilla

FileZilla er en hurtig og pålidelig ftp-klient der virker på flere platforme og understøtter adskillige protokoller og har en intuitiv brugergrænseflade.

Hjemmeside: https://filezilla-project.org
ClamAV

ClamAV is an open source antivirus engine for detecting trojans, viruses, malware & other malicious threats.
Homepage: https://www.clamav.net
Samba

Samba provides file and print services for various Microsoft Windows clients and can integrate with a Microsoft Windows Server domain, either as a Domain Controller (DC) or as a domain member. It supports both Active Directory and Microsoft Windows NT domains.
Homepage: https://samba.org
CSS

De fleste DVD'er er krypteret med CSS (Content Scramble System), der forsøger at begrænse hvilken software der kan afspille en DVD.

SolydXK har ikke de nødvendige pakker til afspilning af krypterede DVD'er præinstalleret.

Når du trykker på 'Installér'-knappen nedenfor, vil følgende pakker blive installeret på dit system:

  • libdvdcss2
  • w32codecs eller w64codecs

Juridisk advarsel:

Kontrollér at gældende lovgivning ikke forhindrer dig i at bruge libdvdcss2 der hvor du er.
