Бизнес приложения

Это список рекомендуемых приложений для бизнеса.
Gnucash

GnuCash — программа финансового учёта. Предназначена для частных лиц и малых предприятий.

Разработанный, чтобы быть простым в использовании, мощным и гибким, GnuCash позволяет отслеживать банковские счета, акции, доходы и расходы. Быстрый и интуитивно понятный в использовании.
Homepage: http://www.gnucash.org
PostBooks

ERP с открытым исходным кодом для продаж, бухгалтерского учета, операций и CRM в единой системе(бухгалтерия, продажи, CRM, закупки и т. д.).

Используйте эту информацию для запуска PostBooks:

  • Имя пользователя: admin
  • Пароль: solydxk
  • Сервер: localhost
  • Порт: 5432
  • База данных: демо, быстрый старт или пустая

Вы можете поддерживать базы данных PostgreSql с помощью pgAdmin III (Login: postgres / solydxk)
Homepage: https://xtuple.com/products/postbooks
ProjectLibre

ProjectLibre это программное обеспечение для управления проектами. Позиционируется как открытая замена Microsoft Project.
Homepage: http://www.projectlibre.org
