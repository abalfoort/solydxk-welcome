Системные приложения

ClamAV

ClamAV — это антивирусный механизм с открытым исходным кодом для обнаружения троянов, вирусов, вредоносных программ и других вредоносных угроз.
Qemu

QEMU (Quick Emulator) is an open-source virtualization software that enables the emulation and virtualization of various operating systems and applications across different hardware platforms.
Homepage: https://www.qemu.org
VirtualBox

VirtualBox — это полноценный виртуализатор общего назначения для оборудования x86, предназначенный для использования на серверах, настольных компьютерах и встраиваемых системах. Пакет VirtualBox устанавливается в существующую операционную систему хоста как приложение; это хост-приложение позволяет загружать и запускать дополнительные гостевые операционные системы, каждая из которых называется гостевой ОС, каждая со своей собственной виртуальной средой.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client — это быстрый и надежный кроссплатформенный клиент FTP, FTPS и SFTP с множеством полезных функций и интуитивно понятным графическим интерфейсом пользователя.
Homepage: https://filezilla-project.org
Homepage: https://www.clamav.net
Samba

Samba предоставляет службы файлов и печати для различных клиентов Microsoft Windows и может интегрироваться с доменом Microsoft Windows Server либо в качестве контроллера домена (DC), либо в качестве члена домена. Он поддерживает домены Active Directory и Microsoft Windows NT.
Homepage: https://samba.org
CSS

Большинство коммерческих DVD-дисков зашифрованы с помощью CSS (системы шифрования контента), которая пытается ограничить программное обеспечение, способное воспроизводить DVD.

SolydXK не поставляется с предустановленными пакетами, необходимыми для воспроизведения этих зашифрованных DVD.

При нажатии кнопки установки ниже в вашу систему будут установлены следующие пакеты:

  • libdvdcss2
  • w32codecs или w64codecs

Юридическое предупреждение:

Ознакомьтесь с местным законодательством, чтобы убедиться, что использование libdvdcss2 законно в вашем регионе.
