Хардверски драјвери

СолидИксК долази са широком палетом преинсталираних драјвера. За најчешће примене, ти драјвери су довољни али у неким случајевима, боље перформансе се могу добити власничким драјверима произвођача.

The install button will open the SolydXK System Settings application. This is a tool to help you installing these hardware drivers. It will check your hardware and list any available proprietary drivers you can install. You only need to select the recommended driver for your hardware and the application will do the rest.

Тренутно Nvidia, AMD/ATI и Broadcom драјвери се могу инсталирати. ДДМ такође дозвољава инсталацију „ПАЕ“ кернела за вишејезгарне 32-битне системе.
