Системске апликације

Када кликнете на дугме за инсталацију, све изабране апликације биће инсталиране на ваш систем.
VirtualBox

Виртуал бокс је пуни виртуализатор опште намене за x86 хардвер намењен за сервере, кућне рачунаре или уграђене системе. Виртуал бокс пакет се инсталира као апликација на постојећем оперативном систему и омогућава инсталацију „гостујућих“ оперативних система у сопственом виртуалном окружењу.
Почетна страна: https://www.virtualbox.org
FileZilla

ФајлЗила клијент је брз и стабилан FTP, FTPS и SFTP клијент са пуно корисних могућности и интуитивним графичким сучељем.
Почетна страна: https://filezilla-project.org
ClamAV

ClamAV is an open source antivirus engine for detecting trojans, viruses, malware & other malicious threats.
Homepage: https://www.clamav.net
Samba

Samba provides file and print services for various Microsoft Windows clients and can integrate with a Microsoft Windows Server domain, either as a Domain Controller (DC) or as a domain member. It supports both Active Directory and Microsoft Windows NT domains.
Homepage: https://samba.org
CSS

Већина комерцијалних ДВД-а су шифровани помоћу CSS (Content Scramble System), којим се ограничава софтвер којим се пуштају ДВД дискови.

СолидИксК не долази са преинсталираним пакетима за пуштање ових шифрованих ДВД дискова.

Кликом на инсталационо дугме, следећи пакети биће инсталирани на ваш систем:

  • libdvdcss2
  • w32codecs или w64codecs

Законско упозорење:

Проверите да ли је коришћење libdvdcss2 легално по законима у вашој области.
