VirtualBox

VirtualBox est un logiciel de virtualisation généraliste pour matériel de type x86, à destination des serveurs, machines de bureau et à usage embarqué. Le paquet VirtualBox s'installe sur un système d'exploitation hôte comme une application; cette application permet de charger et faire fonctionner des systèmes d'exploitation supplémentaires. Chacun est reconnu comme un système d'exploitation invité, avec son propre environnement virtuel.
Homepage: https://www.virtualbox.org

CSS

La plupart des DVD du commerce sont cryptés avec CSS (the Content Scramble System), ce qui tend à limiter le nombre de logiciels capables de lire un DVD.

SolydXK ne préinstalle pas les paquets nécessaires pour lire ces DVD encryptés.

En cliquant sur le bouton d'installation ci-dessous, les paquets suivants seront installés sur votre système:

  • libdvdcss2
  • w32codecs ou w64codecs

Avertissement légal:

Assurez-vous que les lois en vigueur dans votre pays vous autorisent un usage légal de libdvdcss2.
