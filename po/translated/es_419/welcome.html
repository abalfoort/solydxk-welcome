Gracias por usar SolydXK

SolydXK es un sistema operativo de escritorio de código abierto para empresas, organizaciones y usuarios domésticos. Tanto SolydX como SolydK están basados en Debian estable para garantizar la mayor estabilidad y seguridad.

Hemos creado una lista de programas recomendados para empresas y usuarios domésticos que puedes instalar en las siguientes páginas.

Haga clic en el botón Siguiente para empezar.

Software

Todo el software es creado, mantenido y revisado por la comunidad. Las bases de datos de software llamadas "repositorios" se utilizan para ofrecer y mantener más de 60.000 paquetes de software. Por lo tanto, no es necesario descargar software de sitios web dudosos. Simplemente busca e instala el software que necesita con el Gestor de Software.

Soporte

Si eres nuevo en SolydXK o si te quedas atascado, está la comunidad de SolydXK para ayudarte a resolver tu problema. Puedes buscar en el foro para encontrar si tu problema ya ha sido resuelto o iniciar un nuevo tema para obtener ayuda.
Forum: https://forums.solydxk.com
