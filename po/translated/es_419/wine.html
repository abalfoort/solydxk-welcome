PlayOnLinux

SolydXK viene con una amplia variedad de software. Sin embargo, si no encuentras el software adecuado en nuestro repositorio y necesitas utilizar software de MS Windows, puedes instalarlo utilizando PlayOnLinux. No todo el software puede ejecutarse de esta manera, pero un gran número de software ya está soportado.

Al utilizar este software puede hacer que su sistema sea vulnerable a los virus y al malware de MS Windows. El riesgo de infección no es mayor que bajo MS Windows nativo.

Asegúrese de que sólo instala software de terceros de confianza.
