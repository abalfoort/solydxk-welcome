Systeemapplicaties

ClamAV

ClamAV is een open source antivirus engine voor het detecteren van trojans, virussen, malware & andere kwaadaardige bedreigingen.
Qemu

QEMU (Quick Emulator) is open-source virtualisatiesoftware waarmee verschillende besturingssystemen en toepassingen op verschillende hardwareplatforms kunnen worden geëmuleerd en gevirtualiseerd.
Homepage: https://www.qemu.org
VirtualBox

VirtualBox is een algemene full virtualizer voor x86 hardware, gericht op server, desktop en embedded gebruik. Het VirtualBox-pakket installeert op een bestaand host-besturingssysteem als applicatie. Deze hostapplicatie maakt het mogelijk om extra gastbesturingssystemen, elk bekend als een Guest OS, te laden en uit te voeren, elk met zijn eigen virtuele omgeving.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client is een snelle en betrouwbare cross-platform FTP, FTPS en SFTP client met veel handige functies en een intuïtieve grafische gebruikersinterface.
Homepage: https://filezilla-project.org
Homepage: https://www.clamav.net
Samba

Samba biedt bestands- en afdrukservices voor diverse Microsoft Windows clients en kan worden geïntegreerd in een Microsoft Windows Server domein, als domeincontroller (DC) of als domeinlid. Het ondersteunt zowel Active Directory als Microsoft Windows NT domeinen.
Homepage: https://samba.org
CSS

De meeste commerciële dvd's zijn gecodeerd met CSS (Content Scramble System), dat probeert de software die een DVD kan spelen te beperken.

SolydXK heeft niet de nodige pakketten vooraf geïnstalleerd om deze versleutelde dvd's af te kunnen spelen.

Door op de installatieknop te klikken, worden de volgende pakketten geïnstalleerd:

  • libdvdcss2
  • w32codecs of w64codecs

Wettelijke waarschuwing:

Controleer uw lokale wetten om er zeker van te zijn dat het gebruik van libdvdcss2 voor u legaal is.
