Systemprogramvare

ClamAV

ClamAV er en antivirusmotor med åpen kildekode som oppdager trojanere, virus, skadevare og andre ondsinnede trusler.
Qemu

QEMU (Quick Emulator) is an open-source virtualization software that enables the emulation and virtualization of various operating systems and applications across different hardware platforms.
Homepage: https://www.qemu.org
VirtualBox

VirtualBox er en almenn full virtualiserer for x86 maskinvare, rettet mot server, desktop og innebygget bruk. VirtualBox-pakken installeres på et eksisterende vertsoperativsystem som et program; dette vertsprogrammet tillater flere gjesteoperativsystemer, hvert kjent som et gjesteOS, å starte og kjøre i hvert sitt egne virtuelle miljø.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client er en rask og pålitelig multiplattforms FTP, FTPS og SFTP-klient med masse nyttige funksjoner og et intuitivt grafisk brukergrensesnitt.
Homepage: https://filezilla-project.org
Homepage: https://www.clamav.net
Samba

Samba tilbyr fil- og utskriftstjenester for ulike Microsoft Windows-klienter og kan integreres med et Microsoft Windows Server-domene, enten som domenekontroller (DC) eller som domenemedlem. Samba støtter både Active Directory- og Microsoft Windows NT-domener.
Homepage: https://samba.org
CSS

De fleste kommersielle DVDer er kryptert med CSS (the Content Scramble System), som prøver å begrense hvilken programvare som kan spille en DVD.

SolydXK kommer ikke pre-installert med de nødvendige pakkene for å kunne spille disse krypterte DVDene.

Ved å klikke installer-knappen vil følgende pakker bli installert på systemet.

  • libdvdcss2
  • w32 kodeker eller w64 kodeker

Juridisk advarsel:

Sjekk din lokale lovgivning for å være sikker på at libdvdcss2 er lovlig i ditt område.
