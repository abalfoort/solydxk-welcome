Drivere for maskinvare

SolydXK kommer med et bredt spekter av drivere forhåndsinstallert. Dette er som oftest tilstrekkelig, men i noen tilfeller kan du oppnå bedre ytelse med de proprietære driverne fra produsenten.

The install button will open the SolydXK System Settings application. This is a tool to help you installing these hardware drivers. It will check your hardware and list any available proprietary drivers you can install. You only need to select the recommended driver for your hardware and the application will do the rest.

For øyeblikket kan Nvidia, AMD/ATI og Broadcom-drivere bli installert. DDM tillater også å installere PAE-kjernen for 32-bit multi-kjernesystemer.
