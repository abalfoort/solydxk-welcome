Systemanwendungen

Wenn Sie auf die Schaltfläche"Installieren" klicken, werden alle ausgewählten Anwendungen auf Ihrem System installiert.
VirtualBox

VirtualBox ist eine Virtualisierungsumgebung für x86-Hardware und ist auf Server-, Desktop- und Embedded-Einsatz ausgerichtet. Das Paket installiert sich auf einem vorhandenen Host-Betriebssystem als Anwendung, die dann den Start und die Ausführung zusätzlicher Gastbetriebssysteme ermöglicht, jedes mit seiner eigenen virtuellen Umgebung.
Homepage: https://www.virtualbox.org
FileZilla

FileZilla Client ist ein schneller und zuverlässiger Cross-Plattform-FTP, FTPS und SFTP-Client mit vielen nützlichen Funktionen und einer intuitiven grafischen Benutzeroberfläche.
Homepage: https://filezilla-project.org
ClamAV

ClamAV is an open source antivirus engine for detecting trojans, viruses, malware & other malicious threats.
Homepage: https://www.clamav.net
Samba

Samba provides file and print services for various Microsoft Windows clients and can integrate with a Microsoft Windows Server domain, either as a Domain Controller (DC) or as a domain member. It supports both Active Directory and Microsoft Windows NT domains.
Homepage: https://samba.org
CSS

Die meisten kommerziellen DVDs sind mit CSS (Content Scramble System) verschlüsselt. um die Software, die DVDs wiedergeben, in ihrem Funktionsumfang einzuschränken.

In SolydXK sind die für das Abspielen dieser verschlüsselten DVDs notwendigen Pakete nicht vorinstalliert.

Wenn Sie "Installieren" auswählen, werden die folgenden Pakete auf Ihrem System installiert:

  • libdvdcss2
  • w32codecs or w64codecs

Rechtlicher Hinweis:

Beachten Sie die lokalen Gesetze, um sicherzustellen, dass die Nutzung von libdvdcss2 bei Ihnen rechtmäßig ist.
