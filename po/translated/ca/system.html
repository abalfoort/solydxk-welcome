Aplicacions del Sistema

ClamAV

ClamAV és un antivirus de codi obert per detectar troians, virus i altre programari maliciós.
Homepage: https://www.clamav.net
Qemu

QEMU (Quick Emulator) is an open-source virtualization software that enables the emulation and virtualization of various operating systems and applications across different hardware platforms.
Homepage: https://www.qemu.org
VirtualBox

VirtualBox és un virtualitzador de propòsit general per maquinari x86, orientat a l'ús com a servidor, escriptori o encastat. El paquet VirtualBox s'instal·la en el Sistema Operatiu hoste com una aplicació; aquesta aplicació hoste permet carregar i executar diversos Sistemes Operatius convidats, anomenats Guest OS, cadascun amb el seu entorn virtual.
Homepage: https://www.virtualbox.org
FileZilla

Filezilla Client és un client FTP, FTPS i SFTP multi-plataforma ràpid i segur amb moltes funcions útils i una interfície gràfica d'usuari intuïtiva.
Homepage: https://filezilla-project.org
Samba  

Samba facilita serveis de fitxers i impressió per a diversos clients de Microsoft Windows i es pot integrar amb un domini de Microsoft Windows Server, sigui com un Controlador de Domini (DC) o com un membre de domini. Suporta els dominis d'Active Directory i de Microsoft Windows NT.
Homepage: https://samba.org
CSS  

Molts DVD comercials estan encriptats amb CSS (Content Scramble System), el que limita el programari que pot reproduir un DVD.

SolydXK no incorpora per omissió els paquets necessaris preinstal·lats per reproduir aquests DVDs encriptats.

Bo i clicant el botó d'instal·lació de sota s'instal·laran aquests paquets al vostre sistema:

  • libdvdcss2
  • w32codecs o w64codecs

Avís legal:

Consulteu les lleis del vostre país per assegurar-vos que podeu usar legalment lidvdcss2.
